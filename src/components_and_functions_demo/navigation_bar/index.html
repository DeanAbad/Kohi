Navigation bar with right-aligned items and left-aligned logo

This navigation bar has its items aligned on right side and its logo on left side. It automatically hides the items once the viewport shrinks and reaches a certain width.

ClassDescription
nav-wrapperMakes a container for navigation bar that has responsive margin and padding.
left | rightAlign items to left; and to right.
hide-on-med-and-downHides the items in tablet size or below.
brand-logoMakes a brand logo.
Code
                                            
<nav>
    <div class="nav-wrapper">
        <a href="" class="brand-logo">NavBar</a>
        <ul class="right hide-on-med-and-down">
            <li>
                <a href="">Date and Time Input Picker</a>
            </li>
            ...
        </ul>
    </div>
</nav>
